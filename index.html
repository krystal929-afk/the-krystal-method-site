The method

The process starts with what is already true instead of forcing a personality onto the project.

01

Discover

Collect the real pieces: audience reactions, personality, assets, history, goals, and instinct.

02

Decode

Find the pattern underneath the mess. What keeps repeating? What do people respond to?

03

Define

Name the world: voice, visual direction, positioning, rules, and what does not belong.

04

Build

Create the assets, content, pages, books, apps, templates, documents, and workflows.

05

Evolve

Use real feedback and performance data to sharpen the system without losing the core.
