# The Krystal Method — Brand System

## Color System
- **Black (Foundation/Background):** `#050505`
- **Off Black (Surfaces/Depth):** `#0A0A0D`
- **Warm White (Text/Headlines):** `#F0EADF`
- **Muted Taupe (Secondary Text/Supporting):** `#B6B0A7`
- **Dim Taupe (Subtle Text/Disabled):** `#77716B`
- **Electric Lime (Subtle/Highlights):** `#C6FF2E`
- **Royal Purple (Atmosphere/Glows):** `#3A0CA3`
- **Paper (Tags/Labels/Highlights):** `#EEE8DB`

## Typography
- **Headlines / Display:** `Cormorant Garamond` (Bold / Medium / Regular)
- **Body / UI / Supporting:** `Oswald` (Regular / Medium)

## Visual Language
- Dark. Editorial. Intentional. Real. Raw. Refined.
- High contrast. Negative space. Texture. Neon accents.
- Glows: Combination of lime + purple glows on black.
